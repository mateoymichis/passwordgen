# Generador de contraseñas

## Descripción

¡Bienvenidx a mi proyecto! Este es un generador de contraseñas aleatorio. Permite seleccionar la cantidad y el tipo de caracteres incluidos. Además, se puede copiar al portapapeles.

Este proyecto es uno de los primeros que hice utilizando JavaScript y lo he actualizado con algunas mejoras funcionales y visuales.
